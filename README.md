Avalon 

高性能，可伸缩的Java Tcp服务器架构

1.Avalon基于Akka构建的服务器核心。
2.网络部分使用Netty和Mina，二者可根据使用者习惯进行切换。
3. 服务器分为单服务器和多节点分布式服务。如果使用分布式服务，则分为网关服务，和业务服务器。
Akka使用版本2.11-2.3.9
